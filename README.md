## Simple Ecommerce
Simple Ecommerce using NestJS, MongoDB

## Technologies
- NestJS
- MongoDB
- Mongoose
- JWT
- Docker
- Docker Compose

## Installation

```bash
$ cp .env.example .env

$ yarn install

$ docker compose up -d
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

```
